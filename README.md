WebTundraCar
============

#### Setup client

- git clone https://github.com/playsign/WebTundraCar.git
- cd WebTundraCar/
- git clone https://github.com/realXtend/WebTundra.git
- cd WebTundra/
- git checkout dev

#### Car controls

- Steering: a,d / left,right
- Forward/backward: w,s / up,down
- Brake: space
